Executar programa atraves da classe java "principal.java" dentro do NetBeans. 
Usando banco de dados MySql 8.0, alterar usuario e senha na classe "conexao.java" dentro do pacote util.

Ao executar o projeto usar barra de menu "jogos" > "tratamento de jogos" e informar o placar dos jogos.

Tratar os jogos de acordo com a necessidade utilizando comando de pesquisar, salvar, editar, excluir.
